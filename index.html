BROWSER 3D ENGINE

WebGL · Full Rigid Body Physics · Entity/Component · Pure JS · No Dependencies

▶ LAUNCH DEMO

HOW TO USE

1. Run a local server: python3 -m http.server 8080
2. Open localhost:8080
3. Import the engine in your HTML: import { Engine, Vec3, ... } from './js/engine.js'
4. Call Engine.create('#canvas', { init, update }) and game.start()

MATH

Vec3
js/math/vec3.js
3D vector. add, sub, scale, dot, cross, normalise, lerp, reflect, project, angle. Mutating *Self variants for hot loops.
Quat
js/math/quat.js
Unit quaternion. fromAxisAngle, fromEuler, fromTo, slerp, mul, rotateVec, conjugate, toEuler. getForward/Up/Right axes.
Mat4
js/math/mat4.js
Column-major 4×4 matrix. TRS, lookAt, perspective, orthographic, mul, inverse, transpose, transformPoint/Dir. WebGL-compatible Float32Array.
Ray, AABB, Colour
js/math/primitives.js
Ray: intersect plane/sphere/AABB/triangle. AABB: contains, intersects, sdf, expand, transform. Colour: RGB/HSL/hex, lerp, gamma.

PHYSICS

PhysicsWorld
js/physics/world.js
Main simulation. createBody, removeBody, step(dt). Broadphase O(n²) → spatial hash. raycast, raycastAll, overlapSphere, isGrounded.
RigidBody
js/physics/rigidbody.js
Full 6DOF. mass, restitution, friction, linearDamp, angularDamp. addForce, addTorque, addForceAtPoint, applyImpulse. Sleep system. Gyroscopic correction.
Shapes
js/physics/shapes.js
SphereShape, BoxShape (OBB), CapsuleShape, PlaneShape. Each gives correct AABB and inertia tensor for its mass.
Collision
js/physics/collision.js
Narrowphase: sphere-sphere, sphere-box, sphere-plane, box-box (SAT), box-plane, capsule-plane, capsule-sphere. Returns Contact with normal, depth, points.
Solver
js/physics/solver.js
Sequential impulse with 8 iterations. Normal impulse (non-penetration + restitution). Coulomb friction on 2 tangent axes. Warm starting for stability.
Joints
js/physics/joints.js
BallSocket (3DOF constrained), Hinge (1DOF free + motor + limits), Spring (stiffness + damping + break force), Distance (rigid rod).

RENDERER

WebGL context
js/renderer/webgl.js
WebGL2 context. compileShader, createProgram (auto-caches uniforms/attribs). VBO, IBO, VAO, textures, framebuffers. ShaderProgram wrapper with typed uniform setters.
Mesh
js/renderer/mesh.js
Interleaved vertex buffer [pos, normal, uv]. Built-in generators: box, sphere(rings/sectors), cylinder, plane(subdivisions), capsule. fromArrays for custom geometry.
Shaders
js/renderer/shaders.js
Blinn-Phong lit (4 point lights + directional sun), unlit flat/textured, sky gradient, debug wireframe, shadow depth pass. All GLSL 300 es.
Camera
js/renderer/camera.js
Perspective and orthographic. View/projection/VP matrices. lookAt, rotateYaw/Pitch, moveForward/Right/Up. Frustum culling for AABB and spheres. Screen-to-world ray.
Renderer
js/renderer/renderer.js
begin/end frame. drawMesh with frustum culling. Sky gradient. Lighting setup. Debug overlay: debugLine, debugAABB, debugSphere, debugAxes. Draw call stats.
Material
js/renderer/renderer.js
colour, texture, shininess, metallic, roughness, unlit, transparent. Factory: Material.colour(r,g,b), Material.unlit(colour).

GAME

Entity
js/game/entity.js
Scene object with position/rotation/scale. addComponent, getComponent, removeComponent. Parent/child hierarchy with dirty propagation. start/update/lateUpdate/render lifecycle.
Component
js/game/entity.js
Base class. Override: onAttach, onDetach, start, update(dt), lateUpdate(dt), render(renderer). Built-ins: MeshRenderer, RigidBodyComponent, CameraComponent, ScriptComponent.
Scene
js/game/scene.js
createEntity, findByName, findByTag, findById. Drives entity lifecycle. Only updates root entities (children via hierarchy).
Input
js/game/scene.js
keyboard (key codes), mouse (dx/dy, pointer lock), touch (multi-touch delta), gamepad (axes + buttons). moveX/moveZ/jump/sprint convenience axes.
Game
js/game/scene.js
Main loop. Fixed physics timestep, variable render. Engine.create('#canvas', {init, update}) factory. showPhysics flag for debug overlay.
